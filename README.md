# 🎮 Rock Paper Scissors Game

A simple interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript.

---

## Features

- Play against the computer
- Real-time result display (Win / Lose / Tie)
- Score tracking using localStorage
- Auto Play mode
- Keyboard controls (R, P, S)
- Reset score option

---

##  Game Logic

- ✊ Rock beats Scissors  
- 📄 Paper beats Rock  
- ✂️ Scissors beats Paper  

### Example:
- Paper vs Scissors → Scissors wins  
- Rock vs Scissors → Rock wins  
- Same move → Tie  

---

## ⚙️ How It Works

- The computer randomly selects a move
- Your move is compared using JavaScript logic (`if/else`)
- The result is displayed instantly
- Score is saved in the browser (localStorage)
